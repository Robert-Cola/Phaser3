# Phaser3
Projects made using the Phaser3 JavaScript game engine.

Different projects will be stored on their own branches or in their own folders.
All Projects will have a basic Webpage made using HTML/CSS to host/run the Phaser3 code.
An exmplae of this webpage is shown on the main branch in index.html
